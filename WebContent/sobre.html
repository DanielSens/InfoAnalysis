O QUE É A INFO ANALYSIS?

A Info Analysis é um grupo composto por três desenvolvedores, foi criado por uma iniciativa educacional ofertada pelo centro acadêmico UNIBRASIL.

QUAL É O OBJETIVO?

Nosso objetivo é levantar dados sobre a violência e diferença racial atráves de dados públicos anonimizados, desenvolvendo um Dashboard para analizar alguns tipos de dados:

  • Violência;
  • Empregabilidade;
  • Desigualdade social;
